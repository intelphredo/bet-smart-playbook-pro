| Component | Description |
|-----------|-------------|

| `CookieConsent` | GDPR-compliant cookie consent banner |
| `ResponsibleGamblingBadge` | Floating badge linking to resources |
| `AffiliateDisclosure` | FTC-compliant affiliate link disclosures |
| `AgeVerificationGate` | Optional 18+ age verification modal |

### Usage Example

```tsx

  PredictionDisclaimer, 
  AffiliateLink,
  useCookiePreferences 
} from "@/components/legal";

// Add disclaimer to prediction pages
<PredictionDisclaimer variant="banner" />

// Use affiliate links with disclosure
<AffiliateLink href="https://sportsbook.com" sportsbookName="SportsbookName">
  Sign Up Now
</AffiliateLink>

// Check cookie consent
const preferences = useCookiePreferences();
if (preferences.analytics) {
  // Load analytics scripts
```
